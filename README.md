# SmartPark

This repository contains the complete SmartPark application, including the API backend and the Web frontend.

## Structure

- `/api` - The NestJS backend application.
- `/web` - The Vanilla JS/HTML frontend interface.

## Quick Start

### Running the API

```bash
cd api
npm install
npm run start:dev
```

### Running the Frontend

You can run the frontend by serving the `web` directory, for instance using a simple tool like `live-server` or `http-server`:

```bash
cd web
npx http-server
```

*(See the respective README files in each folder for more detailed instructions)*
